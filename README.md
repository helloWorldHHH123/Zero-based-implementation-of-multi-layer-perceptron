# Zero-based-implementation-of-multi-layer-perceptron
Python | Deep Learning | Chapter 04 Multilayer Perceptron

## 📖 简介

本项目针对《动手学深度学习》的“4.2 多层感知机的从零开始实现”进行学习，并手敲代码、理解和注释，仅限于以后方便查看和学习。


### 核心技术
- **深度学习**: PyTorch
- **数据处理**: Pandas, NumPy
- **可视化**: Matplotlib
